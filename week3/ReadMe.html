Успоредност на вектори AB и CD:
AB || CD ≡ AB ⋅ CD = 0

Симетрала:
Нека точка P е от симетралата на AB. Тогава:
P = (A + B) / 2 + t ⋅ AB ⊥
... или: (PA + PB) ⋅ AB = 0

Медицентър на триъгълник ABC:
(A + B + C) / 3

Ъглополовяща BP на BA и BC:
BP = BA * |BC| + BC * |BA|
... или: BP × (|AB| * BC - |BC| * AB) = 0

Задача:
Да определим дали точка е в многоъгълник.
Проверяваме брой пресичания с произволен лъч започващ от точката...

  • При нечетен брой пресичания, точката е вътре.
  • При четен е извън многоъгълника
